# lynx
## A _very_ minimal audio player

This is a for-fun minimal audio player. It's written as a challenge to replicate something that works like xmms or winamp in Rust.

![screenshot](screenshot.png)

Features:
- playlist

Formats supported:
- wav
- flac
- mp3
- ogg
